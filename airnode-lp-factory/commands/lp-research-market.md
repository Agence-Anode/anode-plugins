# Skill /lp-research-market — Contexte marche & audience

## Usage
```
/lp-research-market [client] [audience] [mots-cles]
```

## Instructions

Tu recherches le contexte marche et les pain points de l'audience cible pour orienter le copywriting de la LP.

### Recherches Perplexity (MCP — `perplexity_research`)

1. **Tendances marche** :
   - "Quelles sont les tendances actuelles du marche [secteur] en France en 2025-2026 ?"
   - "Quels sont les enjeux majeurs pour les [audience] dans [secteur] ?"

2. **Pain points audience** :
   - "Quels sont les problemes principaux des [audience] concernant [sujet] ?"
   - "Quelles frustrations les [audience] expriment sur [sujet] ?"

3. **Vocabulaire et intent** :
   - "Comment les [audience] recherchent-ils des solutions pour [probleme] ? Quels termes utilisent-ils ?"
   - Croiser avec les mots-cles AdWords fournis par le consultant

4. **Objections courantes** :
   - "Quelles sont les hesitations principales avant d'acheter [type de solution] ?"
   - "Quels freins a l'adoption de [type de solution] ?"

### Informations a synthetiser

- **3-5 pain points majeurs** de l'audience (avec formulation concrete)
- **Tendances marche** qui renforcent l'urgence (reglementation, evolution techno, concurrence)
- **Vocabulaire natif** de l'audience (les mots qu'ILS utilisent, pas le jargon marketing)
- **Objections a lever** dans la LP (prix, complexite, changement, ROI...)
- **Triggers d'achat** : qu'est-ce qui declenche la recherche de solution ?

### Output

Sauvegarder dans `~/Documents/[client]/assets/recherches/contexte-marche-[date].md`

### Validation

Presenter au consultant :
- "Voici les insights marche pour [audience] :"
- Lister les pain points identifies
- Montrer le vocabulaire cle
- Demander : "Ca colle avec ta vision du terrain ? Des nuances a apporter ?"
